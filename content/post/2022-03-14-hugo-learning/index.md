# Hugo 安装

Hugo 可以运行在多种环境中，比如 macOS、Windowss、Linux、FreeBSD等等。
同样可以托管在喜欢的服务器或者 CDN 上。

- macOS

```bash
$ brew install hugo
```

- Windows

```bash
$ choco install hugo -confirm
```

- Linux

```bash
$ snap install hugo
```
