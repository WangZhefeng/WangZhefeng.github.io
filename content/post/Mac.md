Mac 快捷键：

- `shift + command + g`: 前往文件夹

- `command + space`:  聚焦搜索

- `shift + command + h`: zfwang or home

- `shift + command + a`: 滴答

- `shift + command + d`: 桌面

- `command + ↑`：上层文件夹
- `command + n`：访达
- `shift + command + n`：新建文件夹
- `command + t`: 新建标签页
- `shift + command + backspace`: 清空废纸篓
- `command + backspace`：删除
