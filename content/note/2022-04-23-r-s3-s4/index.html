R 语言的面向对象编程

  • R 中的所有东西都是对象
  • R 支持封装(encapsulation)
  • R 类是多态(polymorphic)的
  • R 允许继承(inheritance)

S3 类

S3 类

S3 类的实例是通过构建一个列表的方式来创建的，这个列表的组件是该类的成员变量。 类属性通过 attr 或者 class 函数手动设置，然后再定义各种泛型函数的实现方法

  • R 中原始的类结构，即 S3 类
  • 大多数 R 中内置的类都是 S3 类的
  • 一个 S3 类包含：
    • 一个列表
    • 一个类名属性
    • 调度(dispathc)功能：使泛型函数(generic function)的使用变成可能
