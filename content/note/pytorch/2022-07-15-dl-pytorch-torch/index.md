## torch.nn 内容

torch.nn 包含了构建神经网络计算图的基本模块:

* Containers
* Convolution Laysers
* Pooling Layers
* Padding Layers
* Non-linear Activations
* Normalization Layers
* Recurrent Layers
* Transformer Layers
* Linear Layers
* Dropout Layers
* Sparse Layers
* Distance Functions
* Loss Functions
* Vision Layers
* Shuffle Layers
* DataParallel Layers
* Utilities
* Quantized Functions
* Lazy Modules Initialization
